# shalculator
A shell-centric calculator with a with a LISP-like scripting interface. 

# Usage
```sh
sub $(pow $(add 3 4 5) 3) -50

# = 1778.000000
```

# Installation
```sh
git clone https://github.com/durcor/shalculator
cd shalculator
# Installs to /bin/ by default
sh ./install.sh
```
